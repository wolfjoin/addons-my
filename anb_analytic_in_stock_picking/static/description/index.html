Analytic Account in Stock Picking

Add an analytic account to incoming/outgoing pickings.

Description

Analytic real time valuation

Since analytic account is included in incoming/outgoing pickings, the stock journal item (generated by the picking transfer) will allow to account incomes and costs against that analytic account.

There are two different ways of stock valuation:
  • Periodical (manual): with this method, users must periodically do a stock regularization by hand, which also means doing an inventory adjustment and its respective journal item of stock regularization. With this module, this journal entry can account incomes and costs against the selected analytic account.

  • Real Time (automated): with this method, Odoo automatically generates a journal entry per stock move transferred. With this module it is possible to specify an analytic account in the stock picking, so the journal entry mentioned above will include this analytic account and therefore it will account the respective incomes and costs. This method gives users information about inventory valuation in real time and its associated incomes and costs, so it is highly recommended and makes more sense when this module is installed.

Installation

To install this module, you need to:
  • You must have installed or available to install the module stock_account.

Usage

To use this module, you need to:
  • First, set the following configuration:
    • Check that your current user has Analytic Account access rights.
    • Go to Settings > Warehouse > Accounting > Accounting, and set Generate accounting entries per stock movement to True (only if you are going to use the real time method).
    • Go to Sales > Products > Products, select the product you are going to use in the picking (it should be stockable).
    • Go to its Accounting tab.
    • Below the Inventory Valuation section, choose Periodical or Real Time in Inventory Valuation field depending on which method you are going to use (with this module Real Time makes more sense).
    • Open the internal category which the product belongs to.
    • In the category, fill in the fields Stock Input Account, Stock Output Account and Stock Valuation Account.
  • Go to Warehouse > Operations > All Operations, and create a picking.
  • Fill in the analytic account field.
  • Transfer the picking.
  • Go to Invoicing > Journal Entries > Journal Entries, and open the journal entries generated by the picking.
  • Check that they have received the analytic account from the picking.
